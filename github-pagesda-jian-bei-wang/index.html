Github Pages+Ghost搭建博客

Ghost是一款静态个人博客模板，利用Github Pages，可搭建免费个人博客。

主要步骤

步骤一：本地架设Ghost环境

Ghost使用buster生成静态页面，故要首先安装buster：

pip install buster

下载Ghost博客框架安装文件，其内部的package.json指定了npm的依赖文件。使用进行依赖库的安装，之后即可启动Ghost本地环境，以在本地进行博客撰写，访问地址为：http://localhost:2368/ghost。

npm install
npm start

根据提示操作进行设置和撰写之后，博客的静态内容实际上还未生成，需在运行ghost server的情况下使用以下命令生成对应的静态页面：

buster setup
buster generate --domain=http://localhost:2368

后续每次进行博客撰写，均需调用buster generate生成静态内容。

步骤二：上传内容到github

Github pages最终用于部署的有效文件为标准的html页面，所以，每次仅将Ghost目录下static文件内容上传至github对应源即可。

首先，需在github上创建github pages的仓库。

然后，使用以下命令将本地static目录与github上新创建的仓库关联，并将新内容一并推送。

cd static
git init
git remote add origin git@github.com:USER/USER.github.io.git
git add --all
git commit -m "Ghost Ready"
git push -u origin master -f

最后，在每次发布新文章后，使用配置好的buster命令将更新推送：

buster generate --domain=http://localhost:2368
buster deploy

步骤三：支持公式

MathJax是在网页中支持Latex公式书写的常用插件，Ghost支持在管理台中直接添加页面的插件代码，使用以下代码即可引入MathJax：

<script type="text/javascript" async  
src="https://cdn.mathjax.org/mathjax/latest/MathJax.js?config=TeX-MML-AM_CHTML">  
</script>

但为了使用行内的公式，需要使用以下代码作为补充：

<script type="text/x-mathjax-config">  
   MathJax.Hub.Config({
     tex2jax: {inlineMath: [['$','$'], ['\\(','\\)']],
     processEscapes: true}          
   });
</script>

步骤四：支持评论

本博客使用了友言社会化评论插件，采用了直接修改content/themes/casper目录下的post.hbs实现。具体地，在//main[@id='site-main']/div[@class='inner']的尾部添加了以下代码：

<div id="uyan_frame"></div>
<script type="text/javascript" src="http://v2.uyan.cc/code/uyan.js?uid=YOUR_USER_ID"></script>
